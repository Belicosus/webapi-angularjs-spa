webapi-angularjs-spa
====================

A reference app for building single page apps (SPA) with ASP.NET Web API 2 and AngularJS.

Technologies
============

Server-Side
-----------

* ASP.NET Web API 2
* EF 6 Code First 
* AutoMapper
* Autofac

Client-Side
-----------

* AngularJS 1.2.4
* Mocha.js, Chai.js, Sinon.js
* Grunt, Bower
